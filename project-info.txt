PROJECT: Power BI Employee Survey Analysis
=========================================

This repository contains a comprehensive Power BI project analyzing employee survey data.

FILES INCLUDED:
- Power BI Project.pbix (Main dashboard and reports)
- Real_Data_Survey_Taken_By_Employees.xlsx (Source data)
- images_powerbi_project/ (Supporting images and assets)

ANALYSIS INCLUDES:
- Employee satisfaction metrics
- Department-wise analysis
- Trend analysis over time
- Key performance indicators

TOOLS USED:
- Microsoft Power BI Desktop
- Microsoft Excel
- Data visualization techniques
- Statistical analysis methods

PROJECT STRUCTURE:
The project follows standard Power BI development practices with separated data sources, transformation logic, and presentation layers.

USAGE:
Open the .pbix file in Power BI Desktop to view the complete analysis and interactive dashboards.
